# 🤖 AI Interview Coach

An AI-powered Interview Preparation Platform built using the MERN Stack, Google Gemini AI, and Puppeteer.

The application analyzes a candidate's resume and job description, generates an AI interview report, suggests interview questions, identifies skill gaps, creates a preparation roadmap, and allows users to download an ATS-friendly resume in PDF format.

---

## 🚀 Features

- User Authentication (Register / Login)
- Resume Upload (PDF)
- AI Interview Report Generation
- Technical Interview Questions
- Behavioral Interview Questions
- Match Score Analysis
- Skill Gap Detection
- AI Preparation Roadmap
- ATS-Friendly Resume Generation
- Resume PDF Download (Puppeteer)
- Modern Responsive UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Adiya-Barswal/Ai-interview-coach.git
```

### Go to Project Folder

```bash
cd Ai-interview-coach
```

---

## Backend Setup

```bash
cd Backend
npm install
npm run dev
```

---

## Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

### Backend (.env)

```env
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
GOOGLE_GENAI_API_KEY=YOUR_GEMINI_API_KEY
```

### Frontend (.env)

```env
VITE_API_URL=http://localhost:3000
```
